# EduSemi-v4x Book  
**― 半導体物性から SystemDK×AITL まで体系的に学ぶ総合設計テキスト ―**

本書 *EduSemi-v4x Book* は、半導体工学・デバイス物性・回路設計・EDA・実習環境構築・先端技術・そして FSM×PID×LLM による知能制御実装までを体系的に学べる、全 34 章構成の総合教材です。

## 基礎編（Chapter 01–08）
1. 第1章：半導体物性とMOS構造の基礎
2. 第2章：デジタル論理と論理回路設計
3. 第3章：プロセス技術と設計限界の理解
4. 第4章：MOSトランジスタ特性と設計基盤
5. 第5章：SoC設計フローとEDAツール
6. 第5a章：仕様策定・モジュール選定・インターフェース設計
7. 第6章：テスト・パッケージ・製品化
8. 第7章：デザインレビューと開発組織連携

## 応用編（Chapter 09–19）
9. メモリ技術の構造と選定指針
10. 高耐圧デバイス
11. ESD設計
12. レイアウト設計と最適化
13. アナログ／ミックスドシグナル
14. 0.18μm AMS設計技法
15. 1/fノイズ半減のアナログ差別化
16. PDKとEDA環境
17. 自動化と実装検証技術
18. FSM設計
19. PLLとクロック設計

## 実践編（Chapter 20–27）
20. 環境構築とツールセット
21. Python自動化ツール
22. Sky130実験とSPICE特性
23. OpenLaneデジタル設計実習

25. 設計評価とレポート
26. SPICE実践演習

## 特別編（Chapter 28–34）
28. 先端ノード技術（FinFET, GAA, CFET）
29. チップレットと先端パッケージ
30. SystemDKの熱・応力・ノイズ設計
